# Roadmap

## Known Issues

- [ ] **Nested drawer not filled** - Some inputs (e.g., Owner) open a new drawer when clicked. The Magic button only fills the current drawer and does not handle nested/child drawers that appear as a result of clicking an input.

## Planned Improvements

- [ ] Auto-detect and fill nested drawers (drawer opened from within a drawer)
- [ ] Support radio button groups
- [ ] Support checkbox fields
- [ ] Support rich text editors (e.g., Quill, TinyMCE)
- [ ] Support file/image upload fields
- [ ] Add a "Magic All" mode that recursively fills nested drawers
- [ ] Add configurable test data profiles (e.g., Vietnamese, English, Japanese)
